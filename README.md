# ROS2-configuration-tentakai-emg-controlled-low-cost-prosthetic-arm
Low-Cost EMG Controlled Prosthetic Arm (ROS2 + MoveIt)

Project Overview

This project presents a ROS2-based simulation and control framework for a low-cost prosthetic arm (Tentakai). The system focuses on kinematic modeling, modular design, and intuitive control using robotics tools.

The implementation enables:

Simulation of a prosthetic arm using URDF modeling
Control of multi-joint chains
Inverse kinematics using MoveIt2

Key Features
Multi-link chain 
Inverse kinematics implementation using MoveIt2
Interactive visualization and control through RViz
Modular design using STL-based components
Compatibility with ROS2 Humble
